## 函數
$$f(x)=(0.64*\sqrt{\lvert x\rvert}-0.8+1.2^{\lvert x\rvert}*cos(200*x))*\sqrt{cos(x)}$$

---

## Script
```python
def f(x):
    return (0.64 * np.sqrt(np.abs(x)) - 0.8 + (1.2**np.abs(x)) * np.cos(200 * x)) * np.sqrt(np.cos(x))
x = np.linspace(-np.pi/2, np.pi/2, 1000)
y = f(x)

# 設定顏色梯度 (使用 plasma colormap)
colors = cm.plasma(np.linspace(0, 1, len(x)))

# 建立圖表
fig, ax = plt.subplots(figsize=(8, 5), facecolor="#1e1e1e")  

# 設定背景色
ax.set_facecolor("#1e1e1e")

# 繪製帶有漸變色的線條
for i in range(len(x) - 1):
    ax.plot(x[i:i+2], y[i:i+2], color=colors[i], linewidth=1 + 2 * (i / len(x)))  # 線條逐漸變粗

# 移除座標
ax.set_xticks([])
ax.set_yticks([])
ax.spines["top"].set_visible(False)
ax.spines["right"].set_visible(False)
ax.spines["left"].set_visible(False)
ax.spines["bottom"].set_visible(False)
```
