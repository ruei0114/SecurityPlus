## 緊繃由訓練解決
1. 想像在馬路邊故寵物
	- 當確定牽繩最長時，寵物也不可能衝到馬路上(大腦清楚肌肉的使用與控制)，可以放任自由奔跑(肌肉不緊繃)
	- 反之當不確定寵物是否有可能跑到馬路(大腦對於肌肉的使用與控制不夠清楚)，便會加強繩子的控制(緊繃)
2. 理論
	- 增加肌肉的氧氣流通量
	- 盡可能在能力範圍使用最大活動度進行運動

## 深蹲動作髂腰肌不適
| problem | solution                                    |
| :-----: | :------------------------------------------ |
|   蹲太低   | 右手放鬆，想像左手折右手食指，剛好碰到的感覺(不刻意向下施壓)<br>用這樣的感覺蹲  |
|   站姿    | 雙腳向外 $30^\circ$ ，與肩同寬就好<br>(站太寬，用關節支撐，意義不大) |
|   站不穩   | 檢查:<br>1. 下蹲是否過快<br>2. 腳趾主動扣緊地面<br>3. 不刻意下蹲 |

> [!info] Note
> 活動度是越大越好沒錯，不過必須在肌肉能力足夠的前提下要求活動度
